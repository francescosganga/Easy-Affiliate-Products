== Description ==
 
Insert Amazon Products in your content using Easy Affiliate Products (EAP) shortcode, with Add-To-Cart system with Amazon's 90-day cookie.

USAGE

Search for ASIN of the Amazon Product:
https://www.amazon.it/Xiaomi-Redmi-Note-Blue-64gb/dp/B07PB8TYCJ/
ASIN Code for this product is "B07PB8TYCJ" (without quotes).

Add EAP shortcode like this: [eap asin="B07PB8TYCJ"]

You can add multiple asin's and set width for each box like this:
[eap asin="B07PB8TYCJ,asin2,asin3" width="33"] -> 33 is 33% for each box.

Enjoy your new Wordpress Plugin.

== Installation ==

1. Upload folder inside zip 'easy-affiliate-products.zip' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Easy Affiliates Products, set Operating Country, your Amazon Tag, your Amazon API and  the API Key and save changes.
